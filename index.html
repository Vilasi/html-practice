A span is a generic element that can be wrapped around something. Like this <p>Lorem ipsum dolor<span>etca</span> mongo spaghet</p>

  • A span is inline. It won't receive its own block like a div does.

Semantic Elements

Semantic elements are tags which separate the page. Like main, header, footer, section, article, aside etc.

  • <article> is used for discrete compositions. Things that can be reused - like a weather widget, or an op ed, or a film review format, etc.
  • <aside> is for things that are meant to be off to the side of the main bit of the page. Or a bit of text, whatever.
  • Header. Header.
  • You can have more than one header, or more than one footer, or more than one aside, etc.
  • Think of semantic elements as boxes to put stuff into.
  • <time></time> is used for denoting the time, date. This one is necessary.
  • <figure></figure> Something like a photo with a description.
  • <abbreviation>
  • <data>
