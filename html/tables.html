<table></table> Child Elements

  • <td></td> : Table Data Cell. Represents a single cell in a table - contains data.
  • <tr></tr> : Table Row - Defines a row of cells in a table.
  • <th></th> : Table Header - Defines the heading of a table column. The top headings are placed withing a <tr></tr> as they will make up their own row of cells.

    • <thead></thead>, <tbody></tbody>, and <tfoot></tfoot</> Describe which part of the data table is which. Useful for accessability and stuff.

  • <thead></thead> <b>:</b> Header
  • <tbody></tbody> <b>:</b> Main Body
  • <tfoot></tfoot> <b>:</b> Footer
